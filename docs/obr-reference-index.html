Owlbear Rodeo Documentation

Comprehensive documentation for Owlbear Rodeo, a virtual tabletop platform. Includes user guides for basic features (rooms, scenes, maps, fog, etc.) and developer documentation for building custom extensions.

User Guides

Learn how to use Owlbear Rodeo and its core features.

  • Getting Started: Introduction to Owlbear Rodeo and basic setup.

  • Rooms: How to create, edit, and invite players to a room.

  • Assets: Managing tokens, maps, and other assets.

  • Scenes: Managing scenes and the infinite canvas.

  • Aligning a Map: Aligning grids for uploaded maps.

  • Images: Working with maps, characters, and image elements.

  • Fog: Hiding parts of your scene using fog of war.

  • Text: Using the rich text editor on the canvas.

  • Drawing: Drawing maps, shapes, and measuring tools.

  • Measure: Measuring distances and spell effects.

  • Permissions: Managing player permissions and roles.

  • Casting: Casting the map to a local screen or TV.

  • Managing your Subscription: Details on billing and subscription tiers.

  • Migration: Migrating data from older versions.

  • Troubleshooting: Common issues and their solutions.

APIs

The core APIs available through the Owlbear Rodeo SDK (@owlbear-rodeo/sdk).

  • APIs Overview: Introduction to the SDK APIs.

  • Action: Manage extension actions shown in the top left of a room.

  • Assets: Interface with the asset manager.

  • Broadcast: Send ephemeral messages to other players in the room.

  • Context Menu: Extend the context menu for selected items.

  • Interaction: Manage custom viewport interactions.

  • Modal: Display modal windows within Owlbear Rodeo.

  • Notification: Send toast notifications to users.

  • Party: Access information about other players connected to the room.

  • Player: Access and update information about the current player.

  • Popover: Display custom UI popovers over the interface.

  • Room: Access room metadata and permission settings.

  • Theme: Access the current theme (light/dark mode).

  • Tool: Create custom interactive tools for the toolbar.

  • Viewport: Manipulate the user's viewport, camera, and zoom.

Scene API

Interact with the infinite canvas, items, and the game scene.

  • Scene Overview: Base scene API for readiness and metadata.

  • Fog: Manipulate the Fog of War.

  • Grid: Access and manipulate grid size, type, and scale.

  • History: Access the undo/redo history stack.

  • Items: Query, add, update, and delete items on the scene.

  • Local: Interact with temporary local items seen only by the current user.
